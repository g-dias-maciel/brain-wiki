## 🔄 Update Instructions

### Step 1: Collect Data from Sheets
1. Open Personal Financial Tracker: https://docs.google.com/spreadsheets/d/1M7Yu3iYCIwWDIwpikPnJdeAuhofulWTW2nanDUD7VdY/edit
2. Open Prop Firm Trading Tracker: https://docs.google.com/spreadsheets/d/19wQeEIIOaf2SaEc1KdIdfAxlUuVwCRsxrbqNOiZqg-k/edit
3. Fill in the template above with current values

### Step 2: Update Wiki Pages
Update the following pages with collected data:

1. **[[Personal Financial Tracking]]**:
   - Update income metrics table with current values
   - Update expense metrics table with current values  
   - Update savings & investment metrics table

2. **[[Prop Firm Trading]]**:
   - Update KPI table with trading performance metrics
   - Update income contribution table
   - Add any new insights from trading journal

3. **[[Financial Dashboard]]**:
   - Update income tables with current amounts
   - Update cash flow summary
   - Update portfolio value if available

4. **[[Business Operations Dashboard]]**:
   - Update prop firm trading revenue in financial integration table
   - Update revenue tracking metrics

### Step 3: Log the Update
Add entry to [[log.md]]:
```
## [YYYY-MM-DD] data-update | Monthly Financial Data Update
Updated financial data from trackers. Current status: [brief summary].
```

## 📋 Monthly Review Questions

### Personal Finance
1. What is the current savings rate compared to target?
2. Which expense categories have increased/decreased significantly?
3. How is income diversification progressing?
4. What adjustments are needed for next month's budget?

### Trading Performance
1. Is trading meeting profitability targets?
2. How does risk-adjusted performance compare to benchmarks?
3. What lessons from the trading journal should be implemented?
4. Should trading capital allocation be adjusted?

### Integration Questions
1. How does trading income compare to business income stability?
2. What is the optimal time allocation between trading and business?
3. How are trading tax implications being managed?
4. What synergies exist between trading skills and business decisions?
