## 🎯 Overview

A comprehensive personal wiki system managed by LLM agents for life organization, business tracking, tax optimization, and strategic planning. This system follows the **LLM Wiki Pattern** for structured knowledge management.

## 🏗️ System Architecture

```
Gabriel Brain/
├── raw/                    # Source documents (immutable)
│   ├── financial/         # Bank statements, tax documents, investments
│   ├── legal/             # Contracts, legal documents, residency
│   ├── business/          # Business plans, invoices, companies
│   ├── personal/          # Journal entries, health records, notes
│   └── assets/            # Downloaded images and attachments
├── wiki/                  # LLM-generated wiki pages
│   ├── life-domains/      # Major life areas
│   ├── entities/          # People, organizations, institutions
│   ├── concepts/          # Strategies, frameworks, methodologies
│   ├── sources/           # Source summaries and metadata
│   ├── dashboards/        # Status overviews and metrics
│   └── outputs/           # Generated reports, analyses, plans
├── tasks.md              # Mobile-accessible task dashboard
├── index.md              # Catalog of all wiki pages
├── log.md                # Chronological activity log
└── CLAUDE.md            # LLM agent instructions and schema
```
